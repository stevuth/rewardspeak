# **App Name**: Rewards Peak

## Core Features:

- User Authentication: Secure sign-up, login, and logout functionality using Supabase Auth.
- Points Dashboard: Display total points, daily earnings, and task progress.
- Offer Walls: Placeholder integrations for offer walls (AdGate, TimeWall, OfferToro, CPALead).
- Task and Survey Sections: UI sections to display available quick tasks, surveys and popular offers.
- Referral System: Enable users to invite others and earn rewards through referral links.
- Withdrawal Requests: Allow users to request withdrawals of earned points via integrated payment placeholder (PayPal/Crypto), to be reviewed on the admin panel.
- SEO Optimization Tool: An AI tool that reviews meta tag content such as titles and descriptions for the best keyword ranking.

## Style Guidelines:

- Primary color: Vivid blue (#29ABE2) evoking trust and openness.
- Background color: Light blue (#E0F7FA), a gentle tint of the primary color, fostering a calming user experience.
- Accent color: A saturated cyan (#00FFFF) to highlight key actions and elements, drawing user attention.
- Body and headline font: 'Inter', a sans-serif font for a modern and neutral feel.
- Mountain-themed icons with a modern, minimalist style.
- Rounded cards and soft shadows to create depth and a friendly interface.
- Animated counters for points and subtle progress bar animations.
